Privacy Policy — Easy Copy Network Info

Last updated: April 2, 2026

Overview

Easy Copy Network Info is a Chrome extension that captures and displays network requests from the current tab for debugging and development purposes.

Data Collection

Easy Copy Network Info does not collect, transmit, or store any personal data to external servers. All data remains locally on your device.

  • Network requests (URLs, headers, payloads, responses) are intercepted only from the active tab and stored locally using Chrome's storage API.
  • User preferences (filter settings, theme) are stored locally on your device.
  • No data is ever sent to any external server or third party.

Permissions

  • sidePanel — Required to display the network inspector panel.
  • storage — Required to save captured requests and preferences locally.
  • tabs / activeTab — Required to interact with the current tab.
  • host_permissions (all_urls) — Required to intercept network requests on any website.

Third-Party Services

Easy Copy Network Info does not integrate with any analytics, advertising, or tracking services.

Changes to This Policy

We may update this policy as the extension evolves. Changes will be reflected on this page with an updated date.
